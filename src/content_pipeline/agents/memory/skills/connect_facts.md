# Connect Facts

## Base Prompt
Find and create connections between isolated facts in the knowledge graph.

Scan for:
1. Unlinked nodes that share common attributes
2. Clusters of high-performing content without explicit links
3. Cross-domain correlations (e.g., topic X + day Y = high ER)
4. Causal chains: what leads to what

Create edges with relation types and confidence weights.
