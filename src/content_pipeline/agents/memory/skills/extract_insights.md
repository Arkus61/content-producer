# Extract Insights

## Base Prompt
Analyze accumulated run data for expert {expert_id} and extract structured insights.

Look for:
1. Patterns: what hooks/topics/structures consistently score high
2. Anti-patterns: what consistently fails
3. Temporal: best days/times for engagement
4. Evolution: how metrics change over time
5. Gaps: areas with insufficient data

Output JSON with insights, patterns, and suggested skill updates.
