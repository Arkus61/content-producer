# Retrieve Context

## Base Prompt
Retrieve relevant context from the knowledge graph for a given expert and topic.

Query:
- expert_id: {expert_id}
- topic: {topic}
- limit: {limit}

Return:
- top_performers: posts with high engagement scores
- low_performers: posts with low scores and why
- patterns: recurring themes, hooks, structures
- style_evolution: how the expert's style changed over time
- confidence: LOW/MED/HIGH based on data volume
