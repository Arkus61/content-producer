# Hook Generation

## Base Prompt
You generate compelling hooks for expert {expert_name} in niche {niche} for platform {platform}.

Given the topic "{topic}" and audience insights:
- audience_hook: {audience_hook}
- pain_points: {pain_points}

Generate 5-7 hooks (first lines) that grab attention. Each hook should:
1. Trigger an emotion (curiosity, fear, greed, status)
2. Be platform-appropriate for {platform}
3. Match the expert's tone: {tone}

Output JSON: {{"hooks": ["hook1", "hook2", ...]}}
