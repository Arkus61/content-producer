# Content Planning

## Base Prompt
You are a content strategist. Create a {days}-day content plan for expert {expert_name} in niche {niche} for platform {platform}.

Consider:
- Audience: {core_segment} and {mass_segment}
- Pain points: {pain_points}
- Product: {product_name} — {product_description}

Output a JSON array of daily topics with:
- day: number
- topic: topic title
- angle: narrative approach
- hook_suggestion: suggested first line
- content_type: "post" or "video"
- goal: awareness / engagement / conversion
